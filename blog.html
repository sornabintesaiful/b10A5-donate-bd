What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents, primarily HTML and XML. It represents the structure of a document as a tree of objects, where each node in the tree corresponds to a part of the document, such as elements, attributes, and text content.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is an event-handling pattern that allows developers to manage events at a higher level in the DOM tree instead of attaching event listeners to individual elements.
This approach is useful because it improves performance by reducing the number of event listeners needed; you only need to set up one event listener on a parent element rather than multiple listeners on child elements. This method simplifies memory management and allows for dynamic handling of new elements added to the DOM.
